About

This is a very simple website to search Pokemon based around the PokeAPI which you can find here

The code used for this website can be found on my Github page

Disclaimer:

  • This is a one page HTML website. This means that the back arrow of your browser won't take you back in the website
  • Since the PokeAPI doesn't have filters for legendaries and pseudo-legendaries, the filtering is done by the website. Thus, those quick searches take a little bit to load. This also means that it only returns legendaries with a base stat total of over 579
  • PokeAPI has some problems with GenVIII Pokemon, so the information page of some of them might not load
  • I added a button to reset the menus because they sometimes glitch
